# arcpy-gis_map_layout-automation

# GIS Automation using ArcPy & ArcGIS Pro

## Project Overview

This project demonstrates GIS automation using Python and ArcPy to programmatically control and standardize map layout elements in ArcGIS Pro.  

It automates repetitive cartographic tasks such as updating map titles, repositioning layout elements (north arrow, scale bar), and exporting final map outputs for reporting and visualization.

This project was developed as part of Digital Earth course at the University of Ottawa lab on geospatial automation workflows.

---

##  Objectives

- Automate GIS map layout customization using Python
- Reduce manual effort in ArcGIS Pro layout design

---

## Tools & Technologies

- Python
- ArcPy (`arcpy.mp`)
- ArcGIS Pro
- GIS Automation
- Cartographic Layout Design

---

##  Workflow Summary

- Loaded ArcGIS Pro project (`.aprx`) using ArcPy
- Accessed layout elements programmatically
- Updated map title dynamically using Python
- Repositioned north arrow and scale bar using coordinate-based layout control
- Exported final layout as PNG for reporting purposes
